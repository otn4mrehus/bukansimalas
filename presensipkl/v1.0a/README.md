## Susunan Direktori
```
+ uploads/
  + presensi/
     - masuk/
     - pulang/
  + lampiran/
     - ijin/
     - sakit/ 
```
### Buat Direktori
```
mkdir -p uploads/presensi/masuk/
mkdir -p uploads/presensi/pulang/
mkdir -p uploads/lampiran/ijin/
mkdir -p uploads/lampiran/sakit/ 
```
### Akses Direktori
```
chmod -R 775 uploads/
```
