# Susunan Direktori
```
+ uploads/
  + foto/
     - masuk/
     - pulang/
  + lampiran/
     - ijin/
     - sakit/ 
```
# Buat Direktori
```
mkdir -p uploads/foto/masuk/
mkdir -p uploads/foto/pulang/
mkdir -p uploads/lampiran/ijin/
mkdir -p uploads/lampiran/sakit/ 
```
# Akses Direktori
```
chmod -R 775 uploads/
```
